# 窗帘知识手册

> 本手册整理自窗帘相关知识，适用于家庭选购与日常维护参考。

## 一、窗帘的作用与功能

### 主要作用
- **保护隐私**：遮挡外界视线
- **调节室温**：减少外界冷热空气影响，提高冷暖效率，降低能耗成本
- **防护家具**：阻挡紫外线，防止家具和地板晒伤褪色
- **美化空间**：通过图案与色彩搭配，塑造房间氛围

### 核心功能

| 功能 | 说明 |
|------|------|
| **遮光** | 调节室内亮度，防止光线外泄，保护隐私 |
| **形状记忆** | 干热处理使褶皱持久，效果维持约3-5年 |
| **光扩散** | 将外部光线均匀扩散，保持室内亮度恒定 |
| **镜面** | 阻挡外部视线，同时切断紫外线 |
| **遮热** | 高密度织物阻隔热气，冬暖夏凉 |

## 二、遮光等级详解

### 白天效果（室内）

| 等级 | 效果 |
|------|------|
| 1级 | 几乎完全黑暗 |
| 2级 | 可识别人物表情 |
| 3级 | 表情可见但较暗 |
| 无遮光 | 允足外部光线进入 |

### 夜晚效果（从外部看）

| 等级 | 效果 |
|------|------|
| 1级 | 无法看到室内 |
| 2级 | 几乎不漏光，看不到阴影 |
| 3级 | 隐约可见阴影 |
| 无遮光 | 阴影清晰可见 |

## 五、窗帘选购指南

### 测量步骤
1. 根据窗帘导轨测量宽度和长度
2. 选择开启方式：大窗口用双开，小窗口用单开
3. 确定悬挂方式（取决于导轨位置）

### 样式对比

| 样式 | 褶皱倍数 | 特点 |
|------|----------|------|
| 2倍 | 导轨宽度×2 | 深层褶皱，美观阴影，隔热效果好 |
| 1.5倍 | 导轨宽度×1.5 | 基础款式，褶皱与花纹平衡 |
| 直型 | 无褶皱 | 平面设计，适合大图案或棉麻材质 |

### 安装方式
- **窗台安装**：悬垂式正面安装 / 蕾丝带天花板
- **天花板安装**：窗帘带天花板 / 蕾丝带天花板
- **窗帘盒安装**：同上
- **凹凸墙面**：窗帘带天花板 / 蕾丝正面安装

## 六、窗帘保养

### 日常护理
- 用吸尘器或掸子定期清除灰尘
- 经常擦拭窗帘轨道和窗户玻璃
- 冬季注意结露，防止发霉

### 洗涤步骤
1. 检查洗涤标签
2. 用刷子或吸尘器清除表面灰尘
3. 沿褶皱折叠，放入洗衣网
4. 使用中性洗涤剂，短时间脱水
5. 手工整理表面，悬挂晾干

### 污渍处理
- 立即用中性洗涤剂蘸湿干布擦拭
- 从污渍外缘向中心擦拭
- 禁用漂白剂、厨房清洁剂、消毒剂

## 七、常见问题

**Q：有宠物应选择哪种窗帘？**
A：推荐可机洗或可手洗的窗帘

**Q：如何选择窗帘颜色？**
A：与地板、墙壁、家具色调协调。同色系营造统一感，互补色增添活力

**Q：日式房间适合什么颜色？**
A：米色、绿色等柔和色调，接近榻榻米的自然感

**Q：窗帘寿命多长？**
A：一般3-5年，取决于材质和保养。无法清除的黑点出现时应更换

## 八、地毯保养要点（补充）

- **日常吸尘**：顺毛方向轻吸，避免旋转刷
- **避免暴晒**：紫外线会导致褪色
- **通风除湿**：防止天然材质发霉
- **污渍处理**：立即用纸巾吸干，从外向内擦拭
